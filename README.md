# 🎮 游戏账号交易平台 - AI Agent 项目

> 基于 LangGraph 的智能游戏账号交易信息展示平台

## 📖 项目简介

本项目是一个展示 **LLMOps 工作流编排** 与 **LangGraph 状态机模式** 实践能力的 Demo 项目。

### 核心特性

- 🤖 **智能对话机器人**：基于 LangGraph 的 AI Agent，支持自然语言交互
- 🎯 **意图识别**：智能识别用户意图（搜索游戏/购买账号/闲聊）
- 📊 **游戏账号浏览**：展示平台支持的游戏、服务器、珍稀装备等数据
- 🔄 **状态机架构**：使用 LangGraph 实现多步骤工作流编排
- 🌐 **前后端分离**：现代化 Web 架构，前端 React，后端 FastAPI
