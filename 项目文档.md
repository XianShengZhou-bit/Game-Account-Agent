# 游戏账号交易平台 - 项目技术文档

> 最后更新：2026-05-20

---

## 1. 项目概述

本项目是一个基于 **LangGraph + FastAPI + React** 的游戏账号交易智能平台，提供游戏账号搜索、AI 智能推荐、交易流程自动化等功能。

### 1.1 核心功能

| 功能 | 状态 | 说明 |
|------|------|------|
| 游戏列表展示 | ✅ 已完成 | 支持游戏图标、搜索 |
| 账号搜索筛选 | ✅ 已完成 | 按游戏、价格、区服筛选 |
| AI 智能对话 | ✅ 已完成 | 基于 LangGraph ReAct 循环 |
| 账号详情页 | ✅ 已完成 | 展示账号完整信息 |
| 账号购买流程 | ✅ 已完成 | Agent 驱动的购买流程 |

    """Agent 主入口函数"""
    # 1. 创建 LangGraph 状态图

**核心流程**：
1. 用户输入 → Reasoner 分析意图
2. 调用工具（数据库查询、LLM 调用）
3. 观察结果 → 决定下一步
4. 循环直到得出答案

#### 4.1.2 工具模块 (tools/)

| 工具 | 文件 | 功能 |
|------|------|------|
| 数据库工具 | database_tools.py | search_accounts, get_supported_games |
| LLM 工具 | llm_tools.py | 调用 OpenRouter API |

### 4.2 API 层 (backend/api/app.py)

#### 主要接口

| 方法 | 路径 | 功能 |
|------|------|------|
| POST | /api/chat | Agent 智能对话 |
| GET | /api/games | 游戏列表 |
| GET | /api/games_list | 支持的游戏列表（从 games.txt） |
| GET | /api/servers | 区服列表 |
| GET | /api/rares | 珍稀装备列表 |
| GET | /api/accounts | 账号搜索（支持筛选） |
| GET | /api/accounts/{id} | 账号详情 |
| POST | /api/accounts/{id}/buy | 购买账号 |
| GET | /health | 健康检查 |

## 5. 数据库设计

### 5.1 accounts 表

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INT | 主键 |
| accounts_id | VARCHAR(36) | 账号唯一标识（Snowflake ID） |
| game_name | VARCHAR(100) | 游戏名称 |
| server_area | VARCHAR(50) | 服务器/区服 |
| title | VARCHAR(200) | 账号标题 |
| price | INT | 价格（单位：分） |
| account_level | INT | 账号等级 |
| hero_count | INT | 英雄数量 |
| skin_count | INT | 皮肤数量 |
| rare_items | JSON | 稀有物品列表 |
| status | TINYINT | 状态（0=在售，1=已售出） |
| version | INT | 版本号 |
| created_at | DATETIME | 创建时间 |

### 5.2 games 表

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INT | 主键 |
| game_name | VARCHAR(100) | 游戏名称 |
| description | TEXT | 游戏描述 |
