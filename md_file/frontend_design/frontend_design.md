# 前端设计文档

## 技术栈
- **框架**: React 18 + TypeScript
- **构建工具**: Vite
- **UI组件库**: Ant Design
- **样式**: Tailwind CSS 3

---

## 页面路由结构

| 路由 | 组件 | 说明 |
|-----|------|------|
| `/` | `HomePage` | 首页，展示游戏列表 |
| `/search?game_name=xxx` | `SearchPage` | 搜索结果页 |
| `/account/:id` | `AccountDetail` | 账号详情页 |

---

## 组件划分

### 1. 通用组件
- **Header** - 顶部导航栏（搜索框、Logo）
- **Footer** - 页脚信息
- **Chatbot** - AI助手聊天窗口（全局浮动）

### 2. 首页组件
- **GameCard** - 游戏卡片（图标 + 名称）
- **GameGrid** - 游戏网格布局
- **SearchBar** - 搜索栏组件

### 3. 详情页组件
- **AccountInfo** - 账号信息展示
- **PurchaseButton** - 购买按钮

---

## 交互流程

### 1. 搜索流程
```
用户输入 → 点击搜索 → 跳转到 /search?game_name=xxx → 展示搜索结果
```

### 2. 聊天流程
```
用户输入消息 → 发送到后端 → 后端处理 → 返回响应 → 展示聊天内容
```

### 3. 购买流程
```
点击购买 → 触发AI对话 → 收集手机号+身份证 → 验证通过 → 显示订单号
```

## UI设计参考

### 配色方案
- **主色**: #4080ff（蓝色）
- **背景**: #f5f5f5
- **文字**: #333333
- **边框**: #e8e8e8

### 布局
- 首页：游戏卡片网格展示
- 搜索页：列表式展示
- 聊天窗口：右下角浮动面板

---

## 状态管理
- 使用 React Context 或 Zustand
- 聊天消息存储在全局状态
- 用户输入实时同步

---

## API交互
- 通过 LangGraph SDK 与后端通信
- 使用 WebSocket 保持长连接
- 消息格式：JSON
